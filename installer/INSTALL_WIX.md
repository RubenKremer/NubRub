# Installing WiX Toolset

## Installation Steps

1. **Download WiX Toolset** from https://wixtoolset.org/releases/
   - Download `wix-cli-x64.msi` for the latest version
   - Also download `WixAdditionalTools.exe`

2. **Install WiX Toolset**
   - Run `wix-cli-x64.msi` installer
   - Check "Add WiX to PATH" during installation
   - Run `WixAdditionalTools.exe` and follow the same process

3. **Verify Installation**
   - Open a new command prompt
   - Run `wix --version` to verify installation

## Troubleshooting

- Ensure you have administrator privileges
- Restart your command prompt after installation
- Check WiX documentation: https://wixtoolset.org/documentation/
