# Installing WiX Toolset
## Installation Steps

### 1. Download WiX Toolset

**Option: Official Website (Recommended)**
- Latest releases: https://github.com/wixtoolset/wix/releases
- Download the `wix-cli-x64.msi` installer for the latest version
- Also download the `WixAdditionalTools.exe`

### 2. Install WiX Toolset

1. Run the downloaded `wix-cli-x64.msi` installer
2. Follow the installation wizard
3. **Important**: During installation, make sure to check the option to "Add WiX to PATH" if available
4. Complete the installation
5. Run the downloaded `WixAdditionalTools.exe` and follow throught the same process

## Troubleshooting

### Installation fails
- Make sure you have administrator privileges
- Try running the installer as administrator
- Check Windows Event Viewer for installation errors

### Still having issues?
- Check the WiX documentation: https://wixtoolset.org/documentation/
- WiX community support: https://wixtoolset.org/community/
